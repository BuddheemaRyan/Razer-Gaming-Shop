# 🎮 Razer Gaming Shop

A sleek, dark-themed e-commerce front-end for Razer gaming products — featuring a fully functional shopping cart built with pure HTML, CSS, and JavaScript.

## ✨ Features

- 🛍️ **Product Listing** — Browse a curated selection of Razer mice, laptops, accessories, and more
- 🛒 **Shopping Cart** — Add/remove items with live quantity and total price updates
- 💚 **Razer-themed UI** — Dark background with Razer's signature green (`#00ffcc`) accent color
- 📱 **Responsive Layout** — Adapts to different screen sizes
- 🔗 **Footer Links** — Quick links to Razer's official site, Twitter, and Instagram

## 🚀 Getting Started

No build tools or dependencies required — just open in a browser.

1. **Clone the repository**
   ```bash
   git clone https://github.com/BuddheemaRyan/Razer-Gaming-Shop.git
   ```

2. **Open the app**
   ```bash
   cd Razer-Gaming-Shop
   open index.html
   ```
   Or simply double-click `index.html` to open it in your default browser.
